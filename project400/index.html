Manual

Add:

Press add to add a new node to the simulation.

Move:

Click and drag a node to reposition it.

Menu:

Click on a node to see it's menu.

Links Menu:

The links menu can be accessed from the Paths menu by clicking on the "<" button next to the "Paths" title. This table displays information about which Nodes are lined to the current node. Weights can also be adjusted from this menu.

Paths Menu:

The paths menu can be accessed from the Links menu by clicking on the ">" button next to the "Links" title. This menu shows the forwarding table generated by the simulation. The table includes all of the Nodes accessible by the current node, how far away that node is, and where to forward traffic to get to that node.

Connect:

Click connect to add a link between nodes. After selecting connect, click on the node you want to establish a link with or on the original node to cancel.

Disable Node:

Click disable to disable the selected node and temporarily remove any links to that node.

Delete Node:

Click delete (under "Functions") to remove the selected node from the simulation.

Weight:

Enter a positive number in the weight box or use the arrows to change the weight of a link.

Disable Link:

Click the "+" or "-" in a link row (in the Links menu under toggle) to enable or disable a link.

Delete Link:

Click "x" in a link row (in the links menu under delete) to remove that link from the simulation.
